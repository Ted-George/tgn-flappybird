# TGN Flappy Bird

Flappy Bird clone built in LÖVE2D (Lua) — CS50 Game Development.
Part of **Teganjo Studios**.

## Features
- Gravity-based bird physics
- Procedurally generated pipe obstacles
- Collision detection and scoring
- Parallax scrolling background

## Built with
- [LÖVE2D](https://love2d.org/) — Lua game framework

## Run locally
1. Install [LÖVE2D](https://love2d.org/)
2. Clone this repo
3. Drag the folder onto the LÖVE executable, or run:love.
